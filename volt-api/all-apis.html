GET /volt/chart Get Chart
▼
Description
Fetch VOLT price or APY chart data for a given time range. type: • price → returns volt_price + usdc_lending_appreciation • apy → returns volt_apy + usdc_lending_apy + target_apy_percent range: • 1m → last 1 month • all → from 2020 to present

Query Parameters
NameTypeRequiredDescription
typestringrequiredprice or apy
rangestringrequired1m (last month) or all (all time)
Example Responses
200 OK
JSON
{
  "type": "price",
  "range": "1m",
  "series": {
    "volt_price": [
      { "date": "2025-03-01", "value": 1.0012 },
      { "date": "2025-03-02", "value": 1.0025 }
    ],
    "usdc_lending_appreciation": [
      { "date": "2025-03-01", "value": 1.000012 }
    ]
  }
}
200 OK
JSON
{
  "type": "apy",
  "range": "1m",
  "target_apy_percent": 6,
  "series": {
    "volt_apy": [{ "date": "2025-03-02", "value": 5.87 }],
    "usdc_lending_apy": [{ "date": "2025-03-01", "value": 4.12 }]
  }
}

POST /withdraw-requests Submit Withdraw Request
Auth Required
▼
Description

NameTypeRequiredDescription
tx_signaturestringrequiredTransaction signature from the Solana blockchain.
volt_amountnumberrequiredAmount of VOLT withdrawn in the transaction.

POST /withdraw-requests/cancel Cancel Withdraw Request
Auth Required
▼
Description
Cancel a pending withdrawal request. The user must first sign a cancel transaction on Solana, then call this endpoint. Note: Only requests not yet COMPLETED, FAILED, or CANCELLED can be cancelled.

Request Body
NameTypeRequiredDescription
tx_signaturestringrequiredTransaction signature of the cancel transaction on Solana.

Request Body
{ "tx_signature": "CANCEL_TX_SIGNATURE_HERE" }
Example Response
200 OK
JSON
{
  "ok": true,
  "id": 1,
  "status": 4,
  "request_id_onchain": "12345",
  "cancel_tx_signature": "CANCEL_TX_SIGNATURE_HERE"
}
