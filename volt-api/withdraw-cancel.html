POST /withdraw-requests/cancel Cancel Withdraw Request
Auth Required
▼
Description
Cancel a pending withdrawal request. The user must first sign a cancel transaction on Solana, then call this endpoint. Note: Only requests not yet COMPLETED, FAILED, or CANCELLED can be cancelled.

Request Body
NameTypeRequiredDescription
tx_signaturestringrequiredTransaction signature of the cancel transaction on Solana.
